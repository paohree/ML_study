# 260326 — 기출 GAP 채우기 (1~3과목)

오늘은 필수노트(기출 반영본)랑 내가 공부한 것(필수노트 copy) 사이의 차이를 수업했다.
4과목은 내 노트에 아예 없어서 처음부터 하기로 했는데 오늘은 시간상 1~3과목 갭만 완료.

---

## 1과목 갭

**5V — Veracity**
3V(Volume/Velocity/Variety)에 Value랑 Veracity가 붙어서 5V.
Veracity = 데이터의 신뢰성/정확성. 데이터가 아무리 많아도 쓰레기 데이터면 결과도 쓰레기니까 (Garbage In Garbage Out).
함정: "유용한 정보 추출" 이러면 틀린 거. 그건 Value임.

**MapReduce 흐름**
Map → 셔플/소트(같은 키끼리 모으기) → Reduce
중간에 셔플/소트 빠뜨리면 틀린 거.

**DW 4특성**
주제지향성 / 통합성 / 시계열성 / 비휘발성
함정: "실시간성"은 DW 특성 아님. 실시간은 OLTP 쪽.

**빅분기 분석방법론 5단계** (CRISP-DM과 별개 개념)
분석 기획 → 데이터 준비 → 데이터 분석 → 시스템 구현 → 평가 및 전개
분석 활용 계획은 모델링 끝나고 세우는 게 아니라 프로젝트 시작 단계부터 수립.

**데이터 품질 4속성**
완전성 / 일관성 / 정확성 / 최신성

**데이터 거버넌스 3요소**
원칙(Principle) / 조직(Organization) / 프로세스(Process)
IT인프라는 포함 안 됨. 도구지 거버넌스 체계 자체가 아님.

**변수의 척도**
| 척도 | 핵심 | 예 |
|---|---|---|
| 명목 | 분류만, 순서/크기 없음 | 혈액형, 성별 |
| 순서(서열) | 순서 있음, 간격 불일치 | 학점, 만족도 |
| 등간 | 간격 일정, 절대 0 없음 | 섭씨온도, 시험점수 |
| 비율 | 간격 일정 + 절대 0 있음 | 키, 몸무게, 매출 |

함정: 섭씨온도 → 비율척도라고 하면 틀린 거. 0도가 "온도 없음"이 아니니까 등간.

**비식별화 기술 6가지**
가명처리 / 총계처리 / 데이터 범주화 / 데이터 마스킹 / 랜덤라운딩 / 제어라운딩
함정: 마스킹 수준이 높을수록 식별이 어려워짐 (쉬워지는 게 아님).

퀴즈: 4/5 (등간척도 문제 틀림 — 몸무게 골랐는데 정답은 섭씨온도)

## 2과목 갭

**상관계수 4종**
| 상황 | 계수 |
|---|---|
| 연속형 ↔ 연속형 (선형) | 피어슨 |
| 연속형 ↔ 연속형 (비선형/이상치) | 스피어만 |
| 이진변수(0/1) ↔ 연속형 | 포인트-비세리얼 |
| 범주형 ↔ 범주형 | 크라메르의 V |

**검정 종류**
| 상황 | 검정 |
|---|---|
| 두 집단 평균 비교 | t검정 |
| 세 집단 이상 평균 비교 | ANOVA(F검정) |
| 범주형 변수 간 독립성 | 카이제곱 |
| 특정 분포를 따르는지 | 적합도 검정 |
| 정규분포 가정 없을 때 | 비모수 검정 |

**중심극한정리**
표본 크기 ≥ 30이면 모집단 분포 관계없이 표본평균이 정규분포에 근사.
함정: "표본 크기와 관계없이 정규분포를 따른다" → 틀린 거.

**불균형 데이터 / SMOTE**
암 5% 정상 95%인 데이터에서 전부 정상 예측해도 정확도 95%가 나오는 문제.
SMOTE = 소수 클래스 데이터를 인위적으로 생성하는 오버샘플링.
함정: "SMOTE는 언더샘플링이다" → 틀린 거.

**변동계수(CV)**
표준편차 ÷ 평균. 단위가 다른 자료를 비교할 때 씀.

퀴즈: 5/5 만점

## 3과목 갭

**Lasso / Ridge / Elastic Net (과적합 방지 정규화)**
- Ridge(L2): 계수를 0에 가깝게 줄임. 완전히 0은 안 됨.
- Lasso(L1): 일부 계수를 아예 0으로 만듦 → 변수 선택 효과.
- Elastic Net: L1 + L2 결합.
함정: "Lasso는 계수를 0에 가깝게만 줄인다" → 틀린 거. 0으로 만들어버림.
변수가 많고 의미있는 게 일부일 때 Lasso가 유리.

**나이브 베이즈**
모든 특성이 서로 독립이라고 가정 (현실에선 비현실적이지만 잘 작동함).
사전확률 + 우도 → 사후확률 계산.
함정: "사전확률과 사후확률로 우도를 계산한다" → 틀린 거. 우도로 사후확률 계산.
텍스트 분류(스팸 필터)에 자주 씀.

**k-fold 교차검증**
데이터를 k개로 나눠서 돌아가며 검증, k번 반복.
vs 홀드아웃: train/test 한번만 나누는 것.
k-fold가 홀드아웃보다 신뢰도 높지만 느림. k 클수록 신뢰도 높음.

**계층적 군집분석**
K-Means와 달리 K 미리 안 정해도 됨.
결과를 덴드로그램(트리 구조)으로 표현. 어디서 자를지 사용자가 결정.
DBSCAN도 K 안 정해도 되고 복잡한 모양도 잡는 밀도 기반 방식.

**PCA (주성분 분석)**
변수들을 조합해서 새로운 축(주성분)을 만드는 차원 축소.
첫 번째 주성분 = 가장 큰 분산 방향.
주성분들 간 상관관계 없음 (서로 직교).

**SVD (특이값 분해)**
PCA랑 비슷하게 차원 축소에 씀.
함정: "정방행렬에만 적용 가능하다" → 틀린 거. 직사각형 행렬도 가능.

**NLP 모델 구분**
- BERT: 양방향으로 문맥 파악. 함정: "단방향이다" → 틀린 거.
- GPT: 대용량 텍스트 사전학습 후 fine-tuning.
- Word2Vec: 단어를 벡터로 변환.
- YOLO: NLP 아님! 실시간 이미지 객체탐지.

**Transformer vs LSTM**
LSTM: 문장을 순서대로 읽음. 긴 문장에서 앞 내용 까먹는 문제.
Transformer: 전체 문장을 한번에 봄. BERT, GPT가 여기 기반.
함정: "Forget Gate는 Transformer 구성요소다" → 틀린 거. Forget Gate는 LSTM.
Transformer 구성요소: Self-Attention, Multi-Head Attention, Positional Encoding.

퀴즈: 5/5 만점
