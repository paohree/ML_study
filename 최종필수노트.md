## ★ 계산형 공식 (무조건 손으로 풀기)

### 베이즈 정리
```
P(질병|양성) = P(양성|질병) × P(질병) / P(양성)
P(양성) = P(양성|질병)×P(질병) + P(양성|정상)×P(정상)
```
유병률 1%, 민감도 90%, 특이도 95%  
→ 0.90×0.01 + **0.05**×0.99 = 0.0585  → 0.009/0.0585 ≈ **15.4%**  
※ 특이도 95% → 위양성률 = **1-0.95 = 0.05**

### z-score / ANOVA F / PCA / F1
```
z   = (x - μ) / σ
MSR = SSR / df_회귀,  MSE = SSE / df_잔차,  F = MSR / MSE
누적기여율 = (λ1+λ2+...+λk) / Σλ
F1  = 2×P×R / (P+R),  P = TP/(TP+FP),  R = TP/(TP+FN)
```
신경망: Σ(입력×가중치) + 편향 → 활성화함수  
ReLU: 음수→0, 양수→그대로

### odds / RR
```
odds = p / (1-p)    ← 1-p 자체는 오답!
RR   = 노출군 발생률 / 비노출군 발생률
```

## 2과목 핵심

**MCAR**: 완전 무작위 (기기 오류) / **MAR**: 다른 변수가 원인 (남성이 우울증 안 씀) / **MNAR**: 결측값 자체가 원인 (소득 높을수록 소득 안 씀)

**결측 처리**: 이상치 있으면 → 중앙값 / 범주형 → 최빈값  
**다중대치법**: 여러 데이터셋 생성 → 결과 결합 (이웃값 대치 = KNN)  
**Hot-deck**: 유사 케이스의 **실제값** 사용 (KNN은 평균)  
**Box-Cox**: 양수 데이터에만, 정규분포 변환  
**SMOTE**: 소수 클래스 **오버샘플링** (언더샘플링 아님)

**왜도**: 오른쪽 꼬리 → **최빈값 < 중앙값 < 평균**  
**IQR** = Q3-Q1 / 이상치: Q1-1.5×IQR 미만 or Q3+1.5×IQR 초과

**상관계수**: 피어슨(등간/비율) / 스피어만(**서열**, 이상치) / 포인트-비세리얼(이진↔연속) / 크라메르V(범주↔범주)  
**편상관**: 제3변수 통제 후 순수 상관 / **정준상관**: 두 변수 **집합** 간 상관

**표본추출**:
- **층화**: 층 내 **동질**, 층 간 이질 → 각 층에서 일부
- **군집**: 집락 내 **이질**, 집락 간 동질 → 집락 통째로

**점추정 4조건**: 불편성(기댓값=모수) / 효율성(분산최소) / **일치성(n↑→모수수렴)** / 충분성(정보최대)  
**표본분산 n-1**: 불편추정량 확보 (계산 편의 아님)

**1종 오류(α)**: H0 참인데 기각 / **2종 오류(β)**: H0 거짓인데 채택 / **검정력 = 1-β**  

|  | H0 참 | H0 거짓 |
|---|---|---|
| 기각 | 1종오류(α) | **검정력(1-β)** |
| 채택 | 1-α | 2종오류(β) |

**가설검정**: 유의수준은 상황마다 다름 ("항상 95%" 틀린 선지)  
**중심극한정리**: n≥30이면 어떤 분포든 정규 근사 ("연속형에만" / "항상 성립" 둘 다 틀린 선지)

**범주형 검정 (카이제곱)**:
- **적합도**: 1변수, 이론분포 비교
- **독립성**: 2변수 1표본, 관련 있나?
- **동질성**: 여러 집단, 분포 같나?

**비모수 검정**:
- 2집단 독립 → **Mann-Whitney U**
- 전후 비교 → **Wilcoxon signed-rank**
- **3집단 이상 → Kruskal-Wallis** ← 자주 혼동

## 3과목 핵심

**지도**: 분류·회귀 / **비지도**: 클러스터링·차원축소  
**로지스틱**: 이름은 회귀지만 **분류** / odds=p/(1-p) / logit=log(odds)

**Lasso(L1)**: 계수 **0으로** 만듦, 변수선택 / **Ridge(L2)**: 계수 **0에 가깝게**  
**SVM**: 마진 최대화, 하이퍼파라미터 최적화 **필요** ("불필요"가 틀린 선지)  
**결정트리**: **비모수**, 정지규칙 필요

**KNN**: **Lazy learning** (사전 학습 없음), k 작으면 과적합  
**앙상블**:
- **배깅**: 병렬, 복원추출 → 랜덤포레스트
- **부스팅**: 순차, 오분류 가중치 → XGBoost (GBM보다 빠름)
- **스태킹**: 기본모델 예측 → **메타학습기** 입력 → 최종 예측

**클러스터링**:
- **K-Means**: 가상 평균점, 이상치에 약함
- **K-medoids**: **실제 데이터 포인트**, **이상치에 강함**
- **DBSCAN**: 밀도 기반, K 미리 불필요
- **계층적**: K 미리 불필요, 덴드로그램
- **GMM**: 여러 정규분포 혼합 가정, 확률적 할당
- **SOM**: 비지도, 고차원→저차원 매핑

**PCA**: 가장 큰 분산 방향, 주성분 간 상관 없음, 정규분포 가정 불필요  
**SVD**: **직사각형 행렬도** 가능 ("정방행렬만" 틀린 선지)

**시계열**:
- **정상성**: 평균/분산 시간과 무관
- **차분**: 오늘-어제 → 추세 제거
- **ADF**: 귀무=비정상, **p<0.05 → 정상** (직관 반대!)
- **ARIMA(p,d,q)**: p=AR차수(PACF), d=차분횟수, q=MA차수(ACF)
- **ACF 절단 → q**, **PACF 절단 → p**

**변수선택법**: 전진(추가) / 후진(제거) / 단계적(둘 다)

**연관분석**: 지지도(같이 구매) / 신뢰도(A 사면 B도) / **향상도>1이면 연관 있음**

**NLP**: BERT(양방향) / GPT(대용량 사전학습) / Word2Vec(벡터변환) / **YOLO=이미지 객체탐지 (NLP 아님!)**  
**Transformer**: Self-Attention / **Forget Gate=LSTM** (Transformer 아님)  
**Seq2seq**: 인코더→컨텍스트벡터→디코더

**딥러닝 활성화함수**: ReLU(은닉층) / Sigmoid(출력, 이진) / Softmax(출력, 다중분류)  
**Vanishing Gradient**: Sigmoid 원인, **ReLU** 해결 / 클리핑 = Explosion 방지

**k-fold**: 모든 데이터 훈련+검증 1번씩 / 홀드아웃보다 **느림**  
**Bootstrap**: 복원추출, 0.632 Bootstrap (63.2% 훈련, 36.8% 테스트)

## 4과목 핵심

**혼동행렬**:
```
Precision = TP / (TP+FP)
Recall    = TP / (TP+FN)
F1        = 2×P×R / (P+R)   ← 분류 전용, 회귀 불가
오분류율  = 1 - Accuracy    ← FP/(FP+TN)이 아님!
```

**ROC**: X=**1-특이도(FPR)**, Y=민감도(TPR) / AUC=0.5은 **무작위 수준** (더 낫지 않음)  
**PR곡선**: **불균형 데이터**에서 ROC보다 유용  
**실루엣**: 범위 **-1~1**, 1에 가까울수록 좋음 (0~1 아님!)  
**Kappa**: 우연 보정 분류 지표, 범위 **-1~1**

**회귀 지표**:  
MAE(이상치 덜 민감) / MSE(이상치 민감) / RMSE / **MAPE(% 오차, 실제값=0이면 불가)**  
**R²**: 독립변수 추가 시 **항상 증가하거나 유지** (감소 안 함)  
**Adjusted R²**: 쓸모없는 변수 추가 시 감소 가능

**VIF**: **독립변수 간** 다중공선성 / **VIF > 10** 의심 ("VIF=종속↔독립" 오답)  
**잔차 자유도**: n-k-1 (n-1은 표본분산 자유도)

**Q-Q plot**: 정규성 확인용 / 직선 = **정규분포** (**균등분포 아님!**) / 오른쪽 위로 굽음 = **양의 왜도** / R² 계산과 무관  
**박스플롯**: 중앙값O, IQR O, **평균 모름**, 최빈값 모름

**파라미터**: 학습으로 결정(가중치) / **하이퍼파라미터**: 사람이 설정(학습률, K) / **Grid Search**: 하이퍼파라미터 탐색

**과적합 방지**: 데이터 **늘리기** O / 데이터 **줄이기** → 악화!  
**편향-분산**: 단순모델=편향↑분산↓ / 복잡모델=편향↓분산↑

**카토그램**: 통계값에 따라 **지역 크기 왜곡** / **단계구분도(Choropleth)**: 색깔 진하기  
**히스토그램**: **연속형** 전용 (범주형 아님)

**시계열 교차검증**: 시간 순서 유지 (무작위 섞으면 안 됨)  
**Lift**: 모델 적중률 / 전체 기저율 (>1이면 모델이 랜덤보다 나음)
