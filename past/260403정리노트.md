## 1. 계산형 — 손으로 풀 수 있어야 함

### 베이즈 정리 (4회 Q13 틀림)
```
P(질병|양성) = P(양성|질병) × P(질병) / P(양성)
P(양성) = P(양성|질병)×P(질병) + P(양성|정상)×P(정상)
```
예) 유병률 1%, 민감도 90%, 특이도 95%  
→ P(양성) = 0.90×0.01 + 0.05×0.99 = 0.009 + 0.0495 = **0.0585**  
→ P(질병|양성) = 0.009/0.0585 ≈ **15.4%**  
※ 특이도 95% → 위양성률(거짓양성) = 1-0.95 = 0.05

### ANOVA F값 (3·4회 반복)
```
MSR = SSR / df_회귀
MSE = SSE / df_잔차
F   = MSR / MSE
```
예) SSR=450, df=3 / SSE=300, df=20  
→ MSR=150, MSE=15, **F=10**  
※ df는 표에서 그냥 읽기 (n-k-1 직접 계산 불필요)

### F1 score (4회 Q33 못품)
```
Precision = TP / (TP+FP)
Recall    = TP / (TP+FN)
F1        = 2 × P × R / (P + R)
```
예) TP=80, FP=10, FN=20  
→ P=80/90≈0.889, R=80/100=0.8  
→ F1 = 2×0.889×0.8/(0.889+0.8) ≈ **0.84**

### PCA 누적기여율 (1회 Q27 틀림)
```
누적 기여율 = (λ1+λ2+...+λk) / Σλ
```
예) λ=6,3,2,1 → 1~2번째 누적 = (6+3)/(6+3+2+1) = 9/12 = **75%**

### 상대위험도(RR) / 승산비(OR) (4회 Q16)
```
RR = 노출군 발생률 / 비노출군 발생률
OR = (노출군 odds) / (비노출군 odds)
odds = p / (1-p)
```
예) 흡연자 40/100, 비흡연자 10/100  
→ RR = 0.4/0.1 = **4**

### odds 계산 (3·4회 반복)
```
odds = p / (1-p)
```
p=0.8 → odds = 0.8/0.2 = **4**  
**1-p(=0.2)는 오답 함정**

### z-score
```
z = (x - μ) / σ
```
평균70, 표준편차15, x=55 → z = (55-70)/15 = **-1**

### 신경망 출력값 (3회 Q28 틀림)
```
출력 = Σ(입력×가중치) + 편향 → 활성화함수 적용
```
예) x1=2, x2=3, w1=0.4, w2=-0.2, b=0.3, ReLU  
= 2×0.4 + 3×(-0.2) + 0.3 = 0.8-0.6+0.3 = **0.5**, ReLU(0.5)=**0.5**

### Lift 계산
```
Lift = 모델 적중률 / 전체 기저율
```
전체 구매율 10%, 모델 타겟 구매율 20% → Lift = **2**

## 3. 개념 구분 — 계속 헷갈리는 것들

### Veracity vs Value (3·4회 반복)
| | 뜻 | 키워드 |
|---|---|---|
| **Veracity** | 데이터 신뢰성·정확성 | "믿을 수 있나?" |
| **Value** | 분석으로 가치 있는 통찰 | "뭔가 얻었나?" |

### 검정력 4칸 표 (3·4회 반복 틀림)
| | H0 참 | H0 거짓 |
|---|---|---|
| **기각** | 1종오류(α) | **검정력(1-β)** ← |
| **채택** | 1-α | 2종오류(β) |

검정력 = 1-β = "실제로 차이 있을 때 올바르게 잡아낼 확률"  
**①"1-α"는 오답 함정**

### 비모수 검정 — 집단 수 구분 (4회 Q24 틀림)
| 상황 | 검정법 |
|---|---|
| 2집단 독립 비교 | **Mann-Whitney U** |
| 같은 집단 전후 비교 | **Wilcoxon signed-rank** |
| **3집단 이상** 비교 | **Kruskal-Wallis** ← |

**"3집단 이상"이 나오면 무조건 Kruskal-Wallis**

### 중심극한정리 함정 (4회 Q15 틀림)
- "**연속형 데이터에만** 적용된다" → **틀린 선지** (어떤 분포든 OK)
- "항상 성립한다" → **틀린 선지** (n≥30 조건 필요)
- "모집단 분포에 관계없이" → **맞는 선지** (함정 아님)

### 인코딩 4종 (3회 Q16 틀림)
| 인코딩 | 설명 |
|---|---|
| **레이블** | 범주에 번호 부여 (혈액형→0/1/2/3) |
| **원핫** | 0/1 벡터로 변환 |
| **순서형** | 순서 있는 범주에 사용 |
| **타겟** | 종속변수 평균으로 대체 |

### 다중대치법 vs KNN 대치 (4회 Q17 틀림)
- **다중대치법**: 결측 데이터셋 **여러 개** 생성 → 결과 결합 → 불확실성 반영
- **KNN 대치**: 가장 가까운 K개 이웃의 **평균**으로 대치
- **Hot-deck**: 유사 케이스의 **실제값** 그대로 사용

### 스태킹 vs 배깅 vs 부스팅 (4회 Q29 틀림)
| 앙상블 | 방식 |
|---|---|
| **배깅** | 병렬, 복원추출 (랜덤포레스트) |
| **부스팅** | 순차, 오분류에 가중치 (XGBoost) |
| **스태킹** | 기본모델 예측값 → **메타학습기 입력 → 최종 예측** |

④"오답에 가중치" = 부스팅 / ③"메타학습기" = 스태킹

### KNN (4회 Q27 틀림)
- **Lazy learning**: 훈련 단계 없음, 예측 시 거리 계산
- k **작으면** 과적합 (k **크면** 과소적합)
- **②"사전 학습, 빠른 예측"** → 오답 함정

### MAPE (4회 Q31 틀림)
- **백분율 오차** → 단위 다른 모델 비교 가능
- 실제값 0이면 **사용 불가** (분모가 0)
- **④"분류 모델 평가"** → 오답 (MAPE는 회귀 전용)

### 척도 4종
| 척도 | 예 |
|---|---|
| 명목 | 혈액형, 성별 |
| 서열 | 학점, 순위 |
| **구간** | **°C 온도** (절대 0 없음) |
| **비율** | 매출액, 키, 몸무게 (절대 0 있음) |

### 분석 준비도 6항목
**분석업무 / 인력및조직 / 분석기법 / 데이터 / 분석문화 / IT인프라**  
*없는 것: 기업규모, 예산, 데이터 품질 등급*

### 데이터 거버넌스 3요소
**원칙 / 조직 / 프로세스** (IT인프라 없음! 준비도에는 있음)

### 층화 vs 군집 추출
- **층화**: 층 내 **동질**, 층 간 이질 → 각 층에서 일부 추출
- **군집**: 집락 내 **이질**, 집락 간 동질 → 집락 통째로 추출

### Lasso vs Ridge
- **Lasso (L1)**: 계수를 **0으로** 만듦 (변수 선택)
- **Ridge (L2)**: 계수를 **0에 가깝게** (0은 안 됨)

### 배깅 vs 부스팅
- **배깅**: **병렬** → 랜덤포레스트
- **부스팅**: **순차** → XGBoost, GBM

## 4. 핵심 지표 공식

### 혼동행렬
```
Accuracy  = (TP+TN) / 전체
Precision = TP / (TP+FP)
Recall    = TP / (TP+FN)
F1 score  = 2×P×R / (P+R)
FPR(1-특이도) = FP / (FP+TN)
오분류율  = 1 - Accuracy
```

### ROC 곡선
- **X축 = 1-특이도(FPR)**, Y축 = 민감도(TPR)
- AUC=1: 완벽 / AUC=**0.5: 랜덤 수준** (0.5보다 나은 게 아님!)

### VIF (4회 Q35 틀림)
- **VIF > 10** → 다중공선성 의심
- **독립변수들 간** 상관관계 측정 (종속↔독립 관계 아님!)
- **③"종속-독립 간 상관관계"** → 오답 함정

### 실루엣 (4회 Q37 반복 틀림!)
- 범위: **-1~1** (0~1이 아님!)
- **1에 가까울수록** 군집 잘 형성됨

### Kappa
- 범위: **-1~1** (0~1 아님)

### Adjusted R²
- 의미 없는 변수 추가 → R²은 올라가지만 Adjusted R²는 **감소 가능**
- R²은 독립변수 추가 시 **항상 증가하거나 유지** (절대 감소 안 함)

### 잔차 자유도
```
잔차 자유도 = n - k - 1  (k=독립변수 수)
```
*n-1은 표본분산 자유도 (다름!)*

## 5. 시각화

### Q-Q plot (4회 Q40 틀림)
- 정규성 확인용: 점들이 **직선에 가까우면 정규분포**
- 직선(대각선) = **정규분포** (**균등분포 아님!** ← 신규 함정)
- 오른쪽 위로 굽음 = **양의 왜도** (오른쪽 꼬리)
- 왼쪽으로 처짐 = 음의 왜도
- R² 계산에 **사용 안 됨**

### 박스플롯으로 알 수 없는 것
- **알 수 있음**: Q1, 중앙값(Q2), Q3, IQR, 이상치
- **알 수 없음**: **평균, 최빈값**, 표준편차

### 카토그램
- 통계값에 따라 **지역 크기를 왜곡**한 지도
- 색깔 진하기로 표현 = **단계구분도(Choropleth)**

## 6. 암기 항목

### SECI 모델
| 단계 | 방향 |
|---|---|
| 공동화 | 암묵→암묵 |
| **표출화(외재화)** | **암묵→형식** |
| 연결화(결합화) | 형식→형식 |
| 내면화 | 형식→암묵 |

### DW vs 데이터레이크
| | DW | 데이터레이크 |
|---|---|---|
| 데이터 | 정형 only | 정형+비정형 |
| 형태 | ETL 정제 | **원시 그대로** |

### HBase vs HDFS
- **HBase** = 분산 **컬럼 기반 NoSQL**
- **HDFS** = 분산 **파일 시스템** (NoSQL 아님!)

### 중심극한정리
- n **≥ 30**이면 모집단 분포 관계없이 정규분포 근사
- "항상 성립" / "연속형에만" → 틀린 선지

### ADF 검정 (시계열 정상성)
- 귀무가설: "비정상이다"
- **p < 0.05 → 귀무가설 기각 → 정상** (직관과 반대!)

### ACF / PACF
| | AR(p) | MA(q) |
|---|---|---|
| **ACF** | 서서히 감소 | **lag q에서 절단** |
| **PACF** | **lag p에서 절단** | 서서히 감소 |

### 분포 왜도와 대표값
- 오른쪽 꼬리: **최빈값 < 중앙값 < 평균**
- 왼쪽 꼬리: 평균 < 중앙값 < 최빈값

### GMM / K-medoids / SOM
- **GMM**: 여러 정규분포 혼합 가정, 확률적 할당
- **K-medoids**: 실제 데이터 포인트를 중심으로 → **이상치에 강함**
- **SOM**: 비지도, 고차원 → 저차원 매핑
