## 1. 계산형 — 손으로 풀 수 있어야 함

### 베이즈 정리
```
P(불량|양성) = P(양성|불량)×P(불량) / P(양성)
P(양성) = P(양성|불량)×P(불량) + P(양성|정품)×P(정품)
```
예) 불량 1%, 양성판정 95%, 정품인데 양성 2%  
= (0.95×0.01) / (0.95×0.01 + 0.02×0.99) = 0.0095/0.0293 ≈ **32.4%**

### ANOVA F값 (3회 Q23 틀림)
```
MSR = SSR / k
MSE = SSE / (n - k - 1)
F   = MSR / MSE
```
예) SSR=300, SSE=200, k=2, n=23  
→ MSR=150, MSE=10, **F=15**

### PCA 누적기여율 (1회 Q27 틀림)
```
k번째 주성분 기여율 = λk / Σλ
누적 기여율 = (λ1+λ2+...+λk) / Σλ
```
예) λ=6,3,1 → 3번째 기여율 = 1/(6+3+1) = **10%**

### 상대위험도(RR) vs 승산비(OR)
```
RR = 노출군 발생률 / 비노출군 발생률
OR = (노출군 odds) / (비노출군 odds)
odds = p / (1-p)
```
예) A약 500명 중 10명 재발, 위약 500명 중 50명 재발  
→ RR = (10/500)/(50/500) = 0.02/0.10 = **0.2**

### odds 계산 (3회 Q30 틀림)
```
odds = p / (1-p)
```
p=0.8 → odds = 0.8/0.2 = **4**  
**① 0.2는 그냥 1-p → 틀린 함정**

### z-score
```
z = (x - μ) / σ
```
예) 평균 70, 표준편차 15, x=55 → z = (55-70)/15 = **-1**

### 신경망 출력값 (3회 Q28 틀림)
```
출력 = Σ(입력×가중치) + 편향  →  활성화함수 적용
```
예) x1=2, x2=-1, w1=0.3, w2=0.4, b=0.2, ReLU  
= 2×0.3 + (-1)×0.4 + 0.2 = 0.6 - 0.4 + 0.2 = **0.4**  
**④ 0.8은 (-1) 부호 빠뜨린 실수**

### Lift 계산
```
Lift = 모델 적중률 / 전체 기저율
```
예) 전체 구매율 10%, 상위 1000명 타겟팅 시 200명 구매  
→ 모델 구매율 = 200/1000 = 20%  
→ Lift = 20%/10% = **2**

### Lasso vs Ridge (2회 Q22 틀림)
| | 규제 | 특징 |
|---|---|---|
| **Lasso** | L1 (절댓값) | 계수를 **0으로** → 변수선택 효과 |
| **Ridge** | L2 (제곱합) | 계수를 **0에 가깝게만** (0은 안 됨) |

### 배깅 vs 부스팅
| | 방식 | 대표 |
|---|---|---|
| **배깅** | **병렬** (독립) | **랜덤포레스트** |
| **부스팅** | **순차** (오분류 보완) | **XGBoost**, GBM, AdaBoost |

### k-fold vs 홀드아웃 (2회 Q25 틀림)
- k-fold 핵심 장점: **모든 데이터가 훈련+검증에 1번씩 사용**
- 속도: **홀드아웃이 빠름** (k-fold가 k번 반복이라 느림)

### 정준상관 vs 편상관 (2회 Q20 틀림)
- **정준상관**: 두 변수 **집합** 간 상관 ([키·몸무게] vs [수학·영어])
- **편상관**: 제3변수 통제 후 두 변수 간 순수 상관

### 분포 왜도와 대표값 순서
- 오른쪽 꼬리(양의 왜도): **최빈값 < 중앙값 < 평균**
- 왼쪽 꼬리(음의 왜도): **평균 < 중앙값 < 최빈값**
- 왜도 = 0: 세 값 일치

## 4. 핵심 지표 공식

### 혼동행렬
```
Accuracy  = (TP+TN) / 전체
Precision = TP / (TP+FP)   ← 예측 양성 중 실제 양성
Recall    = TP / (TP+FN)   ← 실제 양성 중 맞게 예측
F1 score  = 2×(Precision×Recall)/(Precision+Recall)
FPR(1-특이도) = FP / (FP+TN)
오분류율  = 1 - Accuracy   (FP/(FP+TN) 아님!)
```

### ROC 곡선 (1회 Q32 틀림)
- **X축 = 1-특이도(FPR)**, Y축 = 민감도(TPR)
- AUC=1: 완벽 / AUC=0.5: 랜덤 수준

### Kappa (2회 Q34, 3회 Q33 물음표)
- 우연에 의한 일치를 보정한 분류 성능 지표
- 범위: **-1~1** (0~1 아님!)

### PR 곡선 (2회 Q39 틀림)
- **불균형 데이터**에서 ROC보다 유용
- ROC는 불균형 시 낙관적으로 보일 수 있음

### 잔차 자유도 (2회 Q28 틀림)
```
잔차 자유도 = n - k - 1   (k=독립변수 수)
```
*n-1은 표본분산 자유도 — 다름!*

### 표본분산 (n-1)로 나누는 이유 (2회 Q13 틀림)
n으로 나누면 모분산보다 **작게** 나옴 → 편향  
n-1로 나눠야 기댓값=모분산 → **불편추정량**

### 실루엣
- 범위: **-1~1** (1에 가까울수록 좋음)
- 모든 군집 알고리즘에 적용 가능 (K-Means에만 되는 거 아님)

## 5. 시각화 핵심

### Q-Q plot (3회 Q36 물음표)
- 정규성 확인용: 직선에 가까우면 정규분포
- 오른쪽 위로 굽음 = **양의 왜도**
- **R² 계산에 사용 안 됨** (함정 자주 출제)

### 박스플롯으로 알 수 없는 것 (1회 Q33 틀림)
- **알 수 있음**: Q1, 중앙값(Q2), Q3, IQR, 이상치
- **알 수 없음**: **평균, 최빈값**, 표준편차, 분포 형태

### 카토그램 (1회 Q34 틀림)
- 통계값에 따라 **지역 크기를 왜곡**한 지도
- 색깔 진하기로 표현 = 단계구분도(Choropleth)

| | DW | 데이터레이크 |
|---|---|---|
| 데이터 | 정형 only | 정형+비정형 |

| 규모 | 중간 | 대규모 |

### HBase vs HDFS (1회 Q4 틀림)
- **HBase** = 분산 **컬럼 기반 NoSQL**
- **HDFS** = 분산 **파일 시스템** (NoSQL 아님)

### 중심극한정리 (3회 Q19 물음표)
- 표본 크기 **≥ 30**이면 모집단 분포 관계없이 정규분포 근사
- "모집단이 정규분포여야 한다" → **틀린 선지**
- "항상 성립한다" → **틀린 선지** (n 충분히 커야 함)

### GMM (3회 Q27 물음표)
- 데이터가 **여러 정규분포의 혼합**이라고 가정
- 확률적 할당: "클러스터 A 70%, B 30%"
- 군집 수 K **사전 지정 필요** (계층적 군집은 불필요)

### SGD (3회 Q29 물음표)
- **무작위 소수 샘플**로 기울기 계산 (전체 데이터 아님)
- "항상 배치보다 빠르다" → **틀린 선지** (진동·노이즈 발생 가능)
- "학습률 클수록 좋다" → **틀린 선지** (크면 발산)

### SVD (2회 Q29 틀림)
- **직사각형(m×n) 행렬도 가능** ("정방행렬만 가능"이 틀린 선지)

## 7. 함정 패턴 — 시험장에서 바로 적용

| 함정 선지 | 정답 |
|---|---|
| "항상" 들어가면 의심 | RF 트리 수 많을수록 항상 향상 X |
| "옳지 않은 것은?" 문제 | 맞는 말 고르지 말고 틀린 것 찾기 |
| X축=특이도 | X축=**1-특이도(FPR)** |
| Veracity = 가치 있는 통찰 | Veracity=신뢰성 / 통찰=**Value** |
| °C = 비율척도 | °C = **구간척도** |
| DW = 실시간 갱신 | DW = **비휘발성** |
| 훈련 데이터 줄이면 과적합 방지 | 줄이면 **악화**, 늘려야 방지 |
| R² = 독립변수 추가 시 감소 가능 | R²은 항상 **증가하거나 유지** |
| k-fold가 홀드아웃보다 빠름 | k-fold가 **더 느림** |
| HBase = 분산 파일 시스템 | HBase = **분산 컬럼 NoSQL** |
| Kappa 범위 = 0~1 | Kappa = **-1~1** |
| 오분류율 = FP/(FP+TN) | 오분류율 = **1-Accuracy** |
| Q-Q plot = R² 계산에 사용 | Q-Q plot = **정규성 확인용** |
| 검정력 = α | 검정력 = **1-β** |
| 원핫 = 0/1/2/3 번호 부여 | 번호 부여 = **레이블 인코딩** |
| KDD: 변환 다음에 전처리 | 전처리 → **변환** → 마이닝 |
| 박스플롯으로 평균 알 수 있음 | 박스플롯 = **중앙값**, 평균 모름 |
| PR곡선 = 균형 데이터에 유용 | PR곡선 = **불균형 데이터**에 유용 |
| 앙상블 항상 개별보다 뛰어남 | **항상은 틀림** |
