## 4과목 — 빅데이터 결과 해석

### 혼동행렬 (Confusion Matrix)
|  | 예측 Positive | 예측 Negative |
|---|---|---|
| 실제 Positive | TP | FN |
| 실제 Negative | FP | TN |

- **T/F** = 예측이 맞냐 틀리냐
- **P/N** = 뭐라고 예측했냐

### 분류 평가지표
- **정확도(Accuracy)** = (TP+TN) / 전체
  - 클래스 불균형 시 신뢰 못함 (스팸 5개/정상 95개인데 전부 정상 예측해도 95%)
- **재현율(Recall)** = TP / (TP+FN) — 실제 양성 중 잡아낸 비율 (놓친 게 없냐)
- **정밀도(Precision)** = TP / (TP+FP) — 양성 예측 중 진짜 양성 비율 (헛다리 없냐)
- **F1-Score** = 2 × (Precision × Recall) / (Precision + Recall) — 조화평균
  - 조화평균 쓰는 이유: 하나가 낮으면 확 떨어짐 (그냥 평균은 뻥튀기됨)

### Precision vs Recall 상황별 선택
- 암 진단 (놓치면 안 됨) → **Recall** 중시
- 스팸 필터 (정상 메일 스팸함 처박으면 안 됨) → **Precision** 중시

### 임계값(Threshold) 트레이드오프
- 임계값 올리면 → 까다롭게 찍음 → **Precision ↑, Recall ↓**
- 임계값 내리면 → 많이 찍음 → **Precision ↓, Recall ↑**

### ROC 곡선 & AUC
- ROC: 임계값을 0~100% 바꿔가며 성능 그래프
- AUC = ROC 곡선 아래 면적
  - AUC = 1 → 완벽한 모델
  - AUC = 0.5 → 랜덤 수준 (동전 던지기랑 같음), 버려야 함

### 회귀 평가지표
- **MAE**: 오차 절댓값 평균 — 이상치 덜 민감
- **MSE**: 오차 제곱 평균 — 이상치에 가장 민감
- **RMSE**: MSE의 제곱근 — 단위 복원
- **R²(결정계수)**: 모델이 데이터 변동을 설명하는 비율
  - R² = 1 → 완벽
  - R² = 0 → 평균값만 출력하는 것과 다를 게 없음

### 시각화
| 목적 | 그래프 |
|---|---|
| 연속형 데이터 분포 | 히스토그램 |
| 분포 + 이상치 확인 | 박스플롯 |
| 두 변수 관계 | 산점도 |
| 시계열 추이 | 선그래프 |
| 범주형 데이터 비교 | 막대그래프 |
| 상관관계 행렬 | 히트맵 |
