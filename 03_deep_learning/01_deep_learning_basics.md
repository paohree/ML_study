# 딥러닝 기초 메모

## 왜 지금 딥러닝을 가볍게만 보나?

현재 목표가 테뷸러 데이터 입문이라면, 딥러닝은 우선순위가 높지 않다.  
다만 용어를 아예 모르면 나중에 헷갈리므로 핵심만 정리한다.

## 신경망

신경망은 여러 층을 쌓아 입력을 변환하면서 패턴을 학습하는 모델이다.

구성:
- 입력층
- 은닉층
- 출력층

각 층에는 뉴런과 가중치가 있다.

## 활성화 함수

모델이 단순한 직선만 배우지 않도록 비선형성을 넣어 준다.

대표:
- ReLU
- Sigmoid
- Tanh

## 역전파

예측이 얼마나 틀렸는지 계산한 뒤, 그 오차를 뒤에서 앞으로 전달하며 가중치를 수정하는 과정이다.

한 줄 요약:
- "어디를 얼마나 고쳐야 하는지" 계산하는 절차

## 옵티마이저

가중치를 실제로 업데이트하는 규칙이다.

대표:
- SGD
- Adam

초보자는 보통 Adam부터 접한다.

## 학습률, 배치, 에포크

### 학습률

한 번 업데이트할 때 얼마나 크게 움직일지 정하는 값

### 배치 크기

한 번에 몇 개 샘플을 보고 학습할지

### 에포크

전체 데이터를 몇 바퀴 반복할지

## 테뷸러에서의 딥러닝

테뷸러에서도 MLP, TabNet, FT-Transformer 같은 모델이 있다.  
하지만 입문 단계에서는 아래 흐름이 더 효율적이다.

1. 전처리 이해
2. 선형 모델 이해
3. 트리 기반 모델 이해
4. 그 다음에 테뷸러 딥러닝 보기

## 지금 기억할 것

- 딥러닝은 강력하지만, 테뷸러 입문에서는 기본 선택지가 아니다
- 용어만 익히고 넘어가도 충분하다
- 당장 실습 우선순위는 `scikit-learn + 트리 기반 모델`이다
