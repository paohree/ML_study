## 최우선 추가 필요 항목 (기출 직접 연결)

### 데이터 전처리 5단계 흐름
수집 → 정제 → **통합** → **축소** → 변환
- **축소** = 피처 선택/차원 축소로 데이터 크기 줄임 (**노이즈 제거 아님**)
- 6회 기출: "데이터 축소는 노이즈 제거를 위해 정규화 작업을 한다" → 오답

### 변수 선택 3기법 (2회 기출, 반드시 나옴)
| 기법 | 하위 방법 |
|---|---|
| **필터(Filter)** | 카이제곱, 상관계수, 정보이득 |
| **래퍼(Wrapper)** | 전진선택법, 후진제거법, 단계선택법 |
| **임베디드(Embedded)** | Lasso(L1), Ridge(L2), Elastic Net |

- **모델 선택 기준**: Mallow's Cp, AIC, BIC → 값이 작을수록 좋음
- **주성분 기법은 변수 선택 기법 아님** (2회 기출 함정)

### PCA 주성분 선택 기준 3가지 (3,4,6회 반복 출제)
1. 누적기여율 **85% 이상** (기준 범위 70~90%)
2. 고유값(eigenvalue) **1 이상**인 주성분만 선택
3. **Scree plot** — 꺾이는 지점까지

### Boxplot에서 알 수 없는 것 (2회 기출)
- Boxplot으로 알 수 있는 것: Q1, Q2(중앙값), Q3, IQR, 이상값
- **Boxplot으로 알 수 없는 것: 평균**

### 이상치 처리 3분류 (6회 기출)
- **제거(Deleting)**: 이상치 데이터 삭제
- **대체(Imputation)**: 중앙값/평균 등으로 대체
- **변환(Transformation)**: 오른쪽 꼬리 → 로그/제곱근 변환, 왼쪽 꼬리 → 지수/제곱 변환

### 차원의 저주 (2,3회 기출)
- 차원 증가 시 데이터 공간 희소화, 거리 멀어짐, 모델 정확도 저하
- 해결: 차원 축소(PCA, LDA 등)

## 높은 우선순위

### VIF 판단 기준 수치 (3회 기출)
- VIF > 1이면 의심
- **VIF >= 10이면 다중공선성 있다고 판단**
- 해결: 변수 선택 / Ridge/Lasso / 데이터 추가 수집

### FA vs PCA 차이 (3회 기출)
- **FA(요인분석)**: 잠재 요인 추출 — 원인 파악 목적
- **PCA(주성분분석)**: 분산 보존 — 차원 축소 목적

### MDS (다차원척도법) (2,3회 기출)
- 대상 간 **거리 보존**하며 저차원 배치
- **스트레스 값(Stress Value)**으로 적합도 평가

### 타깃 인코딩 (5회 기출)
- 해당 범주의 **타겟 변수 평균 또는 확률**로 대체
- ❌ "표준편차를 활용한다" → 오답

### 로버스트 스케일링
- 공식: `(x - median) / (Q3 - Q1)`
- 이상치에 강함 (중앙값과 IQR 사용)
- 3회 기출: "스케일링 방법 아닌 것은? → 변수 범주화"

### 요약변수 vs 파생변수 (2,6회 기출)
- **요약변수**: 여러 모델에서 공통 재활용 (총구매금액, 횟수 등)
- **파생변수**: 특정 조건/함수로 생성, 매우 주관적, 논리적 타당성 필요

## 중간 우선순위

### 68-95-99.7 규칙 (정규분포)
- ±1σ = 68%
- ±2σ = 95%
- ±3σ = 99.7%

### 머신러닝 이상치 탐지 기법 (선지 소거용)
마할라노비스 거리, Isolation Forest, LOF(Local Outlier Factor), One-Class SVM

### 차원 축소 기법 전체 목록
| 기법 | 특징 |
|---|---|
| PCA | 분산 보존, 차원 축소 |
| FA | 잠재 요인 추출 |
| MDS | 거리 보존, 스트레스 값으로 평가 |
| SVD | 직사각형 행렬도 가능 |
| LDA | 지도학습 기반 차원 축소 |
| ICA | 독립 성분 분리 |
| t-SNE | 시각화 목적, 2~3차원 매핑 |

### 독립변수/종속변수 별칭 (선지 소거용)
- 독립변수 = 설명변수 = 예측변수 = 입력변수
- 종속변수 = 반응변수 = 출력변수 = 결과변수

### 구간화(Binning)
- 연속형 변수 → 범주형 변수로 변환
- 이상치 영향 줄이는 효과
