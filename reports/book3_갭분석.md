알고리즘(KNN, SVM, 앙상블, 딥러닝)은 잘 정리됨.
**회귀분석 파트(모형 가정, 검정법, 변수 선택, 모형 평가 지표)가 전반적으로 얇음.** 교재에서 가장 많은 분량이고 출제 비중도 높음.

## ★★★ 최우선 추가

### 회귀모형 4가지 가정 + 검정법 매칭
| 가정 | 확인 방법 |
|---|---|
| 선형성 | 잔차 vs 예측값 산점도 |
| 독립성 | **Durbin-Watson 검정** |
| 등분산성 | **Bartlett 검정**, **Levene 검정** |
| 정규성 | Q-Q plot, **K-S 검정**, **Shapiro-Wilk 검정** |

### 변수선택법 3가지
| 방법 | 방향 | 특징 |
|---|---|---|
| **전진선택법** | 없음 → 추가 | 유의한 변수 하나씩 추가 |
| **후진제거법** | 전부 → 제거 | 기여도 낮은 변수 하나씩 제거 |
| **단계별 선택법** | 양방향 | 추가와 제거 번갈아 수행 |

### SST / SSR / SSE 관계
- **SST = SSR + SSE** (전체 = 회귀로 설명된 변동 + 잔차)
- **R² = SSR / SST**
- **F통계량 = MSR / MSE** → 회귀 모형 전체 유의성 검정
- **t통계량** → 개별 회귀계수 유의성 검정

### 최소제곱법 (OLS)
- 회귀계수 추정 원리: **잔차 제곱합(SSE)을 최소화**
- "최소자승법"이라고도 함
- ※ 필수노트의 "MSE 최소화"는 평가지표 표현 — 추정 원리는 OLS/SSE 최소화

### 다중공선성 지표
- **VIF > 10** 이면 다중공선성 있다고 판단
- **CI(조건지수) > 30** 이면 심각한 다중공선성

### AIC / BIC
- 모형 비교/변수 선택 기준 — **작을수록 좋은 모형**
- AIC = -2ln(L) + 2k
- BIC = -2ln(L) + k·ln(n) (n 클수록 복잡도에 더 큰 패널티)

## ★★ 중요

### Holdout + LOOCV
| 방법 | 설명 |
|---|---|
| **Holdout** | 훈련/검증/테스트 분리. 통상 7:3 또는 8:2 |
| **LOOCV** | N개에서 1개만 테스트, N-1개 학습. N번 반복 |
| k-fold | 필수노트 있음 |
| Bootstrap | 필수노트 있음 (0.632) |

### 레버리지 + Cook's Distance
- **레버리지**: 독립변수(x값)이 극단적인 이상치
- **Cook's Distance**: 특정 관측치 제거 시 회귀계수 변화량
- **Cook's D > 1** 이면 영향력 큰 관측치

### L1-norm / L2-norm
- **L1-norm (Manhattan)**: |x₁| + |x₂| + ... → **Lasso**
- **L2-norm (Euclidean)**: √(x₁² + x₂² + ...) → **Ridge**

### 강화학습
- 에이전트가 환경에서 행동 → 보상/패널티 → 보상 최대화 학습
- 지도/비지도/강화학습 3분류 문제 출제

### 분석모형 구축 절차 4단계
요건 정의 → 모델링 → 검증 및 테스트 → 적용

## ★ 미세 보완

### 로지스틱 회귀
- **Odds Ratio** 용어 명시: exp(β) = Odds Ratio
- logit 수식: `log(p/(1-p)) = β₀ + β₁x₁ + ...`

### Lasso/Ridge
- λ(lambda) 역할: **클수록 패널티 강함**, λ=0이면 일반 회귀
- L1/L2-norm 연결

### 스태킹
- 모델 결과를 **메타 모델**이 학습
