## 2과목 패턴

### 빈출 토픽 (양 시험 모두)
- z-score 계산 (평균60 표준편차10 x=70 → z=1)
- 중심극한정리 ("항상 적용" / "연속형만" 함정)
- 가설검정 ("항상 95%" 함정)
- 베이즈 정리 계산 (여성/180cm 이상 동일 문제 반복)
- 결측값 대체 — 회귀분석 대체 시 변수관계 보존
- 왜도/이상치 — summary 해석, Wind max 이상치
- 상관계수 4종 구분
- 인코딩 4종
- SMOTE — 오버샘플링 (언더샘플링 함정)
- 변동계수(CV) — 단위 다른 자료 비교
- 표본분산 자유도 n-1

### test2 추가 출제
- Box-Cox 변환
- 상대위험도(RR)/승산비(OR) 계산
- 1종/2종 오류 구분
- 편상관분석
- 단봉분포 오른쪽 꼬리에서 최빈값 < 중앙값 < 평균

### 함정 패턴
- "항상" 단어 삽입
- 방향 뒤집기: 회귀 대체 → 변수관계 약해진다
- SMOTE를 언더샘플링으로 기술
- 1/n vs 1/(n-1) 불편추정량 혼동

### 계산형

| 베이즈 정리 | O | O(동일) |
| 상대위험도/승산비 | - | O |
| 1종/2종 오류 확률 | - | O |

### 아직 미출제 (출제 가능성 높음)

## 3과목 패턴

### 빈출 토픽 (양 시험 모두)
- NLP — YOLO는 이미지 객체탐지
- Transformer 구성요소 — Forget Gate는 LSTM
- Seq2seq — 인코더→컨텍스트벡터→디코더
- 신경망 출력값 계산
- Vanishing Gradient — 시그모이드 원인, ReLU 해결, 클리핑은 Explosion 방지
- 경사하강법 — SGD=무작위, 학습률 크면 발산
- 의사결정나무 — 비모수, 정지규칙
- SVM — 초매개변수 최적화 필요 없다는 함정
- XGBoost — GBM보다 빠름
- 계층적 군집분석 — 군집 수 미사전지정, 덴드로그램
- 랜덤포레스트 — 나무 많을수록 좋다는 함정
- Lasso/Ridge 수식 구분
- SVD — 정방행렬만 가능 함정
- SOM — 비지도, 저차원 매핑
- k-fold — 홀드아웃보다 느림
- 나이브 베이즈 — 사전+우도→사후 (순서 뒤집기)
- MANOVA — 독립1개이상 + 종속여러개
- 잔차 자유도 — n-p (n-1 함정)
- 부스팅=순차, 배깅=병렬
- PCA — 정규분포 가정 불필요 함정

## 4과목 패턴

### 빈출 토픽 (양 시험 모두)
- ROC 곡선 — X축=1-특이도(FPR), Y축=민감도(TPR)
- 혼동행렬 계산 — Accuracy/Recall/Specificity/Precision
- F1 score — 분류 전용, 회귀에 불가
- 과적합 — 데이터 수 늘리면 방지 (유발 아님)
- 편향-분산 트레이드오프 — 단순모델=편향↑분산↓
- 히스토그램 — 연속형 전용
- 카토그램 — 통계값에 따라 지역 크기 조정 (면적 아님)
- 결정계수(R²) — 독립변수 많을수록 커짐
- 엘보우 메소드 — 최적 k 탐색
- 박스플롯 — 95% 신뢰구간 알 수 없음, **평균도 알 수 없음**
- 로지스틱 회귀 — 잔차 정규성 불필요

### test2 추가 출제
- 앙상블이 항상 개별보다 뛰어나다는 함정
- 다중공선성
- 시계열 교차검증 — 시간 순서 유지
- 인포그래픽 종류 5종
- Q-Q plot — 오른쪽 위로 굽음=양의 왜도

### 함정 패턴
- X축/Y축 뒤집기: ROC "X축=특이도"
- 앙상블 "항상 뛰어나다"
- R² "종속→독립 설명 비율"로 방향 뒤집기
- "데이터 수 늘리면 과적합 유발"

### 계산형
| 문제 유형 | test1 | test2 |
|---|---|---|
| 혼동행렬 지표 | O | O |
| ANOVA 표 빈칸 | - | O |

### 아직 미출제 (출제 가능성 높음)
- MAPE/MSLE/RMSLE/Adjusted R² 구분
- 실루엣 점수 -1~1
