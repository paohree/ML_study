## 학습 플랫폼

단순 Claude 수업 도구를 넘어, **멀티에이전트 기반 자동화 학습 플랫폼**으로 구성되어 있다.

### 에이전트 구조

```
수업 AI (Claude — 메인)
    소크라테스식 수업 진행
    질문으로 끌어낸 뒤 설명, 피드백 반복
         ↓ 수업 단위 완료
평가 AI (Claude — 서브에이전트)
    퀴즈 생성 → 채점 → 약점 분석
    진행상황.md 업데이트 → session_log.md 기록
         ↓ 세션 종료
기록 AI (Codex)
    session_log.md 읽고 → YYMMDD.md 생성
```

### 자동화 (Harness)

| 시점 | 동작 |
|---|---|
| 세션 종료 | 기록 AI 실행 → 변동사항 git commit & push |
| 매일 12시 / 17시 | 변동사항 있으면 자동 git push (cron) |

에이전트 역할 정의는 [`agents/`](agents/) 폴더에서 관리한다.
