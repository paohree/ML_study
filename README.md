정형 데이터 기반 ML 프로젝트에서 전문가 팀원들과 협업할 수 있는 수준을 목표로 하는 개인 스터디 저장소.
Pandas, sklearn, numpy, scipy, XGBoost 등 핵심 라이브러리 실습과 개념 정리를 병행한다.

**단기 목표**: 빅데이터분석기사 필기 합격 (2026-04-04)

---

## 학습 방법

Claude를 전담 교수로 삼아 **소크라테스식 대화형 수업**으로 진행한다.

```
Claude가 질문으로 끌어냄
    → paohree가 생각하고 답함
        → Claude가 설명 + 보완
            → paohree가 자기 말로 다시 설명
                → Claude가 피드백
                    → 퀴즈 3~5문제 풀기
                        → 오답 즉시 재정리
                            → 다음 개념으로
```

단순히 내용을 받아 적는 방식이 아니라, 매 개념마다 "왜 이게 필요한가"를 직접 생각하고 답한 뒤 설명을 듣는 구조다. 퀴즈는 실제 필기시험 형식(객관식/단답형)으로 출제되고, 틀린 문제는 바로 내 말로 재정리한다.
